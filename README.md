Описание программы (Кейс-задача №2 — Счётчик)

Данная программа представляет собой простейший веб-счётчик, созданный с использованием HTML, CSS и JavaScript. Приложение позволяет пользователю изменять числовое значение с помощью двух кнопок – «плюс» и «минус» – и наблюдать визуальное изменение результата.

Назначение программы

Цель приложения — демонстрация работы:

кнопок с обработчиками событий (Event Listeners),

изменения DOM-элементов через JavaScript,

динамического изменения стилей в зависимости от состояния,

ограничений значений,

отображения уведомлений пользователю.

Описание функционала
1. Интерфейс

Пользовательский интерфейс состоит из:

кнопки «плюс» (+)

кнопки «минус» (–)

окна отображения результата (по умолчанию 0)

дополнительного окна для предупреждения

Стилизация реализована с помощью CSS:

фон всей страницы — линейный градиент,

кнопки центрированы по странице и имеют скругление 5 px,

кнопки тоже имеют градиентный фон,

элементы выровнены по центру как по горизонтали, так и по вертикали.

2. Логика работы

Работа приложения реализована на JavaScript.

При нажатии на кнопку “плюс”

– значение счётчика увеличивается на 1.

При нажатии на кнопку “минус”

– значение счётчика уменьшается на 1.

Значение отображается в центральном окне.

3. Изменение цвета фона результата

Программа автоматически меняет цвет окошка результата:

Значение счётчика	Цвет
> 0 (положительное)	жёлтый
< 0 (отрицательное)	зелёный
= 0	красный

Цвет меняется сразу после каждого изменения.

4. Ограничения значений

При достижении предела:

+10 — кнопка «плюс» становится неактивной

−10 — кнопка «минус» становится неактивной

При возврате в диапазон (например, с 9 → 10 → 9):

кнопки снова активируются автоматически.

5. Сообщение о достижении экстремума

Когда значение становится 10 или –10,
внизу появляется надпись:

«Вы достигли экстремального значения»

При возврате в диапазон предупреждение исчезает.

6. Защита и стандартные практики

Также используются стандартные функции взаимодействия с DOM и обработчики событий (addEventListener), что соответствует требованиям по созданию простых динамических веб-страниц.
